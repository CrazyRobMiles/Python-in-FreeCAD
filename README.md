# Python in FreeCAD
Sample Python programs which create interesting objects in FreeCAD.

## Getting started 
* [First Panel](/Examples/01%20FirstPanel/FirstPanel.md) Create your first panel
* [Punched Card](/Examples/02%20Punched%20Card/PunchedCard.md) Create a panel with random holes in it. Like a punched card.
* [Secret Encoder](/Examples/03%20Secret%20Encoder/SecretEncoder.md) Create a set of puzzle panels which fit together to reveal a secret message.
* [Velcro(tm) Cards](/Examples/04%20Velcro%20Cards/VelcorCards.md) Create two unique panels which fit together. 
* [A pair of very matching pendants](/Examples/05%20Pendant%20Pair/PendantPair.md) Create a pair of unique pendants to be shared between significant others. 

## Making Boxes
* [Desk Tidy Box](/Examples/06%20Desk%20Tidy/DeskTidy.md) A simple box desk tidy you can use to, er, tidy up your desk.
* [Desk Tidy Box with Lid](/Examples/07%20Desk%20Tidy%20With%20Lid//DeskTidyWithLid.md) Otherwise known as a box. The lid fits over the box. If you print the lid and the box with different colours they look quite nice. 
* [Box with flush lid](/Examples/08%20Box%20with%20Flush%20Lid/BoxFlushLid.md) A box with a lid that is flush with the sides of the box. The lid is friction fit and stays on well. 
* [Round Boxes](/Examples/09%20Round%20Boxes/RoundBoxes.md) A whole bunch of round boxes with flush lids which fit inside each other. Use them to make a really annoying present for someone. 
* [Lotsa Boxes](/Examples/10%20Lots%20of%20Boxes/LotsOfBoxes.md) Round and square boxes in different sizes. 
* [PICO Box](/Examples/11%20PICO%20Box/PicoBox.md) A box which can house your favourite micro-controller.

## Extrusion 

* [Extruded Panel](/Examples/12%20Extruded%20Panel/ExtrudedPanel.md) A program that makes a panel (just like the very first example) but does it by extruding a rectangle. 
* [Extruded Star](/Examples/13%20Extruded%20Star/ExtrudedStar.md) A program that makes a star with any number of points and then extrudes it.
* [Extruded Vase](/Examples/14%20Extruded%20Star%20Vase/ExtrudedStarVase.md) A vase made from an extruded star which has been hollowed out. 



For each project you get a description, a picture that shows you what it looks like and the Python code. You can open the source files as Python programs using **File>Open** in FreeCAD version 1.0. 
If you want general tips on Running Python in FreeCAD and how to get started, go [here](https://github.com/CrazyRobMiles/freecad-tips)
