# Python in FreeCAD
Sample Python programs which create interesting objects in FreeCAD. These form the examples for a series of articles in [Raspberry Pi Magazine](https://magazine.raspberrypi.com/)

## Getting started - [Raspberry Pi Magazine Issue 161](https://magazine.raspberrypi.com/issues/161)

In this article we get started with FreeCAD and make a simple panel. We then punch some random holes in the panel to generate unique keycards. Then we finish off with more secret stuff; a set of cards which, when stacked together reveal a secret message, two cards which have unique matching pins and sockets and finally a pair of pendants that you can share with a significant other. 

* [First Panel](/Examples/01%20FirstPanel/FirstPanel.md) Create your first panel. (page 70)
* [Punched Card](/Examples/02%20Punched%20Card/PunchedCard.md) Create a panel with random holes in it. Like a punched card or a super secure key that only you and your secret contact have copies of (pages 71 and 72)
* [Secret Encoder](/Examples/03%20Secret%20Encoder/SecretEncoder.md) Create a set of puzzle panels which fit together to reveal a secret message. (page 69 Figure 1)
* [Velcro(tm) Cards](/Examples/04%20Velcro%20Cards/VelcorCards.md) Create two unique panels which fit together. 
* [A pair of very matching pendants](/Examples/05%20Pendant%20Pair/PendantPair.md) Create a pair of unique pendants to be shared between significant others. 

## Making Boxes - [Raspberry Pi Magazine Issue 162](https://magazine.raspberrypi.com/issues/162)

In this article we make some boxes, starting with a simple open box and then adding lids. On the way we see how to create complex shapes just by cutting and fusing. There are a couple of extra example programs. One makes a large number of circular boxes and the other makes a large number of round boxes. 

* [Desk Tidy Box](/Examples/06%20Desk%20Tidy/DeskTidy.md) A simple box desk tidy you can use to, er, tidy up your desk. (page 54)
* [Desk Tidy Box with Lid](/Examples/07%20Desk%20Tidy%20With%20Lid//DeskTidyWithLid.md) Otherwise known as a box. The lid fits over the box. If you print the lid and the box with different colours they look quite nice. (page 55)
* [Box with flush lid](/Examples/08%20Box%20with%20Flush%20Lid/BoxFlushLid.md) A box with a lid that is flush with the sides of the box. The lid is friction fit and stays on well. (page 55)
* [Round Boxes](/Examples/09%20Round%20Boxes/RoundBoxes.md) A whole bunch of round boxes with flush lids which fit inside each other. Use them to make a really annoying present for someone. 
* [Lotsa Boxes](/Examples/10%20Lots%20of%20Boxes/LotsOfBoxes.md) Round and square boxes in different sizes. 
* [PICO Box](/Examples/11%20PICO%20Box/PicoBox.md) A box which can house your favourite micro-controller. (page 56)

## Extrusion - [Raspberry Pi Magazine Issue 163](https://magazine.raspberrypi.com/issues/163)

This article is all about paths and extrusion. Making complex shapes using simple trigonometry. There is a bonus program makes interesting large shapes by combining multiple smaller ones. 

* [Extruded Panel](/Examples/12%20Extruded%20Panel/ExtrudedPanel.md) A program that makes a panel (just like the very first example) but does it by extruding a rectangle. (pages 88 and 89)
* [Extruded Star](/Examples/13%20Extruded%20Star/ExtrudedStar.md) A program that makes a star with any number of points and then extrudes it. (pages 92 and 93)
* [Extruded Vase](/Examples/14%20Extruded%20Star%20Vase/ExtrudedStarVase.md) A vase made from an extruded star which has been hollowed out. (page 93)
* [Multi-Star Vase](/Examples/15%20Multi%20star%20vase/Multistarvase.md) A more complex vase made from several vases which have been squished together.

## Paths and Fonts 

A glimpse into the future. The descriptions will be added when the magazine is released.

* [Rounded Rectangle Path](/Examples/16%20Rounded%20Rectangle%20Path/RoundedRectanglePath.md) A program that creates an outline for a luggage tag with a rounded end using lines and a curve.
* [Tag with Hole](/Examples/17%20Tag%20with%20hole/TagWithHole.md) A program that shows how a shape defined by one path can be subtracted from another. 
* [Tag with relief text](/Examples/18%20Tag%20with%20relief%20text/TagWithReliefText.md) A program that draws relief text on a luggage label. Can be used on a printer with a single coloured filament. 
* [Tag with embedded text](/Examples/19%20Tag%20with%20embedded%20text/TagWithEmbeddedText.md) A program that draws text embedded in the surface of the tag. Produces two printable elements which can be combined to make the tag. You will need a printer that supports multiple filaments. 
* [Tag with internal text](/Examples/20%20Tag%20with%20internal%20text/TagWithInternalText.md) A tag which contains the text embedded just underneath the surface. Works well with white and transparent filaments for the outer case. 
* [Party Tags](/Examples/21%20Party%20Tags/PartyTags.md) Having a party and want to print out a custom name tag for everyone attending? This program will do exactly that.

For each project you get a description, a picture that shows you what it looks like and the Python code. You can open the source files as Python programs using **File>Open** in FreeCAD version 1.0. 
If you want general tips on Running Python in FreeCAD and how to get started, go [here](https://github.com/CrazyRobMiles/freecad-tips)
